# AgriDeal AI - Smart Crop Market Negotiator

AgriDeal AI is a browser-based smart crop selling assistant for farmers. It helps farmers compare market prices, negotiate with buyers, check scam risk, plan storage, and sell directly to homes, hotels, and restaurants.

## Key Features

- AI-style crop price prediction
- Profit calculator and buyer offer analysis
- Smart buyer trust score
- Scam risk detection
- Market shock alerts
- Crop selling heatmap
- Smart selling timer
- Emergency market rescue suggestions
- Smart storage recommendation
- Farm-to-consumer selling mode for homes, hotels, and restaurants
- Support for crops, vegetables, fruits, and flowers
- Multilingual UI support
- Voice input and speech output based on selected language
- Negotiation chat assistant
- Auto-generated deal report

## Tech Stack

- HTML
- CSS
- JavaScript
- Web Speech API
- Static browser deployment

## How to Run

Open `index.html` in any modern browser.

## Suggested GitHub Pages Deployment

1. Create a GitHub repository named `agrideal-ai-smart-crop-market-negotiator`.
2. Upload `index.html`, `styles.css`, `app.js`, `hero-market.svg`, and this `README.md`.
3. Go to repository `Settings > Pages`.
4. Select branch `main` and folder `/root`.
5. Your live project link will be:

```text
https://YOUR_GITHUB_USERNAME.github.io/agrideal-ai-smart-crop-market-negotiator/
```
